ABSTRACT

The Movie Review Website is a robust web application built using the MERN stack (MongoDB, Express.js, React, Node.js) to address the growing need for user-friendly platforms where individuals can discover films and share critiques. This project leverages the performance of Node.js and the dynamic UI capabilities of React to create a fast, responsive, and secure environment for movie enthusiasts. Core functionalities include user authentication, a comprehensive catalog of movies with search functionality, and detailed pages where users can submit ratings and textual reviews. This report details the complete software development lifecycle, encompassing requirements gathering, system design through Entity-Relationship and Data Flow diagrams, database normalization, implementation, and testing methodologies. The resulting platform provides an intuitive community space that successfully facilitates reliable movie discovery and interactive review sharing.
